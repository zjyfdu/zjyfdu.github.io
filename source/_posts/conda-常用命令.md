- 创建虚拟环境

```shell
conda create -n your_env_name python=x.x anaconda
```

最后的anaconda可选，有的话，会安装很多包，numpy、sklearn等等
- 激活

```shell
source activate your_env_name
```

- 安装包

```shell
conda install -n your_env_name [package]
```

- 不激活

```shell
source deactivate
```

- 已经有的虚拟环境

```shell
conda env list
```

- 删除虚拟环境

```shell
conda remove -n your_env_list -all
```

- 检查conda是否安装

```shell
conda -v
```

- 更新conda

```shell
conda update conda
```
