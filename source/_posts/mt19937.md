### mt19927
- [Mersenne Twister](https://en.wikipedia.org/wiki/Mersenne_Twister)是目前比较常用的随机数生成器
- 周期非常长$2^{19937}-1$，速度非常快

### c++代码
```
#include <iostream>
#include <random>

int main()
{
  unsigned seed1 = 1337;
  std::mt19937 g1(seed1);
  for(int i=0; i<100; i++)
    std::cout << 1.0*g1()/g1.max() << std::endl;
}
```
